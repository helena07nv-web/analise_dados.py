# Análise de Dados com Pandas

Este projeto é um exemplo simples de uso da biblioteca pandas em Python.

## O que o projeto faz
- Cria um DataFrame
- Trata valores nulos na coluna de preço
- Cria a coluna valor_total (preço × quantidade)

## Tecnologias usadas
- Python
- Pandas

## Como executar
1. Instale o pandas:
   pip install pandas
2. Execute o arquivo Python
